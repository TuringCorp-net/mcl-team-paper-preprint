# MCL-Team: Validating the Meta-Coordination Layer via Minimal Multi-Model Collaboration on Public Benchmarks

**Preprint**. Companion to the MCL conceptual framework preprint.  
This paper provides the first public benchmark validation of the MCL Team algorithm.

## Abstract (short)
We present a minimal realization of the Meta-Coordination Layer (MCL) — the Team algorithm — and evaluate it on LiveBench and IFEval. A simple pipeline of parallel generation, cross-reference, and fusion achieves state-of-the-art Instruction Following and Language scores while scaling Reasoning from 84.5 to 97.0 across three quality-cost tiers. Results are cross-validated by independent LLM analysis.
