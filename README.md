# MCL-Team: Minimal Multi-Model Collaboration Surpasses Single Models on LiveBench and IFEval

**Preprint**. Companion to the MCL conceptual framework preprint.  
**Canonical DOI**: https://doi.org/10.6084/m9.figshare.32824352

This paper provides the first public benchmark validation of the MCL Team algorithm.

## Abstract (short)
We present a multi-model collaboration algorithm that operationalizes the Meta-Coordination Layer (MCL) framework for public benchmark evaluation. On the LiveBench 2026-01 leaderboard, MCL-Team achieves Language (93.5–94.2) and Instruction Following (87.4–88.8) scores exceeding all single models; Reasoning climbs from 84.5 to 97.0 across three quality-cost tiers. On IFEval, it reaches 92.6% prompt-level strict accuracy and 95.0% instruction-level strict accuracy. A clean ablation isolates +6.3 Reasoning points attributable to coordination depth alone.
